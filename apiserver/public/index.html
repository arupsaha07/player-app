No Route HTTP verb Description / Parameters
1 /api/user/getByPage GET This will return the first 10 user list i.e. page no 1.
2 /api/user/getByPage?pageNo=2 GET This will return the user list as per the given page no.
2 /api/user/get/:id GET This will return the user as per the requested id.
2 /api/user/getAll GET This will return the list of all users
3 /api/user/authenticate POST username , password
4 /api/user/register POST firstname , lastname , username , password
5 /api/user/update POST firstname , lastname , username , password
6 /api/user/delete POST id
